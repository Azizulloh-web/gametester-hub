# gametester-hub
🚀 GameTester Hub
GameTester Hub — это платформа для разработчиков и тестеров. Разработчики публикуют свои проекты для сбора обратной связи, а тестеры оценивают игры, оставляют отзывы и помогают проектам стать лучше.

🛠 Основные функции
Умная фильтрация: Поиск игр по жанрам и статусам разработки (Прототип, Альфа, Релиз и т.д.).

Система отзывов: Возможность поставить оценку и оставить развернутый комментарий с поддержкой редактирования и удаления.

Мультимедиа: Загрузка обложек и галерей скриншотов для визуальной презентации игры.

API: Интеграция с Django REST Framework для передачи данных внешним сервисам.

🏗 Технологический стек
Backend: Python, Django

API: Django REST Framework

Frontend: HTML5, CSS3 (Bootstrap/Custom)

Database: SQLite3 (development)
