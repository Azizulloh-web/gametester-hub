🎮 GameTester Hub
GameTester Hub — это веб-платформа для разработчиков инди-игр и игровых тестеров. Система позволяет разработчикам публиковать свои проекты для получения обратной связи, а тестерам — оценивать игры, загружать отчеты и следить за процессом разработки.

🚀 Основные возможности
Система фильтрации: Удобный поиск проектов по жанрам и статусу разработки (от "Только идея" до "Релиза").

Управление отзывами: Возможность оставлять оценки, редактировать и удалять свои комментарии.

Галерея проекта: Поддержка загрузки обложек и наборов скриншотов геймплея.

Интерактивный интерфейс: Современный дизайн с использованием Django-шаблонов.

API: Поддержка REST API для интеграции с внешними приложениями.

🛠 Технологический стек
Backend: Python 3.x, Django 5.x

API: Django REST Framework

База данных: SQLite3 (development)

Frontend: HTML5, CSS3, JavaScript

📦 Как запустить проект локально
Клонируйте репозиторий:

Bash
git clone https://github.com/ВАШ_НИК/gametester-hub.git
cd gametester-hub
Создайте виртуальное окружение и установите зависимости:

Bash
python -m venv venv
source venv/bin/activate  # Для Windows: venv\Scripts\activate
pip install -r requirements.txt
Примените миграции базы данных:

Bash
python manage.py migrate
Запустите сервер:

Bash
python manage.py runserver
Сайт будет доступен по адресу http://127.0.0.1:8000/

💡 Архитектура
Проект построен на паттерне MVT (Model-View-Template), что обеспечивает четкое разделение данных, бизнес-логики и пользовательского интерфейса.
